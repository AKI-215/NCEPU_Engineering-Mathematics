# 华北电力大学工程数学攻略

## 工程数学基础复习重点 (仅供参考) 
### 一、 矩阵理论部分 (约40%) 
1. 常用矩阵范数 
定理 2.2.5：设 A=(aij)∈Rn×n，则常用矩阵范数定义如下：
- 行范数（∞-范数）：∥A∥∞=max1≤i≤n∑j=1n∣aij∣
- 谱范数（2-范数）：∥A∥2=λ1（其中 λ1是 ATA的最大特征值）
- 列范数（1-范数）：∥A∥1=max1≤j≤n∑i=1n∣aij∣
- Frobenius 范数（F-范数）：∥A∥F=∑i=1n∑j=1n∣aij∣2=tr(AHA)（即 AHA的所有特征值之和的平方根）
2. 矩阵的广义逆（减号逆与加号逆） 
2.1 定义与性质 
- 减号逆 A−：满足 AA−A=A，且 rank(AA−)=rank(A−A)=rank(A)。同时满足幂等性：(A−A)2=A−A，(AA−)2=AA−。
- 加号逆（Moore-Penrose 逆） A+：
- 定义 1.4.1：设 A∈Cm×n，若 G∈Cn×m满足：
  1. AGA=A
  2. GAG=G
  3. (AG)H=AG
  4. (GA)H=GA
  5. 则称 G为 A的加号逆，记作 A+。它是矛盾方程 Ax=b的极小范数最小二乘解对应的广义逆。
2.2 计算方法 
定理 1.4.3：若 A∈Cm×n，且 rank(A)=r>0，A=BC为其满秩分解，则：
A+=CH(CCH)−1(BHB)−1BH
定理 1.4.5：方程 Ax=b的极小范数最小二乘解唯一存在，且为 x=A+b。
例题：求方程组
⎩⎨⎧x1+2x2−x3+2x4=12x1+4x2+x3+x4=5−x1−2x2−2x3+x4=4
的极小范数最小二乘解。
3. 子空间的结构（基与维数） 
例题：设 (α1,α2,α3,α4)是线性空间 V的一组基，子空间定义为：
- V1=span(α1−α2+α4,α2−α3)
- V2=span(α1+α4,α2+α3,α1+α2+α3+α4)
- 求 V1+V2的维数。
另一例题：设 V1=span{1+2t+t2,−1+t+t2+t3}，V2=span{2−t+t3,1−t+3t2+7t3}
(1) 求 V1+V2和 V1∩V2的基与维数。
(2) 判断 V1+V2是否为直和。如果不是，将其基扩充为 R3(t)（三次多项式空间）的基。
4. 矩阵的最小多项式与 Jordan 标准型 
定理 3.3.3：n阶方阵 A的最小多项式 ψ(λ)唯一存在，且等于其特征矩阵 λE−A的第 n个不变因子 dn(λ)。
重点：
- 矩阵 Jordan 标准型的求解（矩阵的秩是关键指标）。
- 矩阵函数的求解（待定系数法，关键弄清函数与矩阵的谱）。
例题：设 A=4−11301−420
(1) 求 A的最小多项式。
(2) 求 A的 Jordan 标准型。
(3) 求 sin(2A)。
解答提示：
(1) ∣λE−A∣=(λ−2)(λ−1)2，且 r(E−A)=2，故最小多项式为 ψ(λ)=(λ−2)(λ−1)2。
(2) Jordan 标准型为 2111。
(3) 设 sin(2A)=a0E+a1(2A)+a2(2A)2，通过待定系数法求解系数 a0,a1,a2。
5. 内积空间中抽象向量的长度与夹角 
核心：利用基下的度量矩阵计算向量的内积、长度和夹角。
例题：设度量矩阵 A=1−100−14300312900912，向量 b=12129。
(1) 设内积空间 V在基 α1,α2,α3,α4下的度量矩阵是 A，求向量 α1−α2+2α3与 2α2−α3+3α4的夹角。
计算过程：设 b1=(1,−1,2,0)T，b2=(0,2,−1,3)T（坐标向量）。
- 内积 (b1,b2)=b1TAb2=35
- 长度平方 ∥b1∥2=b1TAb1=43，∥b2∥2=b2TAb2=70
- 夹角 θ=arccos(43×7035)
6. 线性变换的表示矩阵、值域与核空间 
例题：已知 A=2100130000λ+10000λ(λ+1)。
(1) 若 A是内积空间 V在一组基下的度量矩阵，则参数 λ需满足什么条件？并求向量 α1−α2与 α2+α3+3α4的夹角。
(2) 设 A是线性变换 T在一组基下的表示矩阵。若值域 R(T)的维数为 3，则 λ需满足什么条件？并求核空间 N(T)。
解答提示：
(1) 度量矩阵需正定，故 λ(λ+1)>0且 λ+1>0，解得 λ>0。
(2) R(T)=R(A)，要求 rank(A)=3。当 λ=0时，A的第四列为零，秩为 3。此时核空间 N(T)由方程 Ax=0的解空间生成，即 N(T)=span{α4}。值域 R(T)=span{2α1+α2,α1+3α2,α3}。
7. 矩阵的 LU 分解 
要求掌握将矩阵分解为下三角矩阵 L和上三角矩阵 U的乘积的方法。

二、 数值分析部分 (约40%) 
1. 线性方程组迭代法 
将方程组 Ax=b改写为 x=Bx+f，构造迭代格式：x(k+1)=Bx(k)+f。
- 收敛性判定：
  - 定理 4.1.1：迭代法收敛的充要条件是谱半径 ρ(B)<1。
  - 充分条件：若矩阵范数 ∥B∥<1，则迭代法收敛。
- 收敛速度：定义渐近收敛速度为 R(B)=−lnρ(B)。
2. 非线性方程迭代法 
将方程 f(x)=0化为不动点形式 x=φ(x)，构造迭代格式：xk+1=φ(xk)。
- 收敛性定理（定理 4.3.2）：若迭代函数 φ(x)在区间 [a,b]上满足压缩条件，则对任意初值 x0∈[a,b]，迭代序列收敛。
- 收敛阶判定（定理 4.3.3）：若 φ′(x∗)=0，且 φ(p)(x∗)=0，则迭代法为 p阶收敛。
练习：求解 x2−5=0，迭代格式为 xk+1=xk+c(xk2−5)。
(1) 求使迭代收敛的 c的取值范围。
(2) 求使收敛速度最快的 c值。
3. 函数的最佳平方逼近 
在函数空间 C[a,b]的子空间 Φ=span{φ0(x),…,φn(x)}中，寻找 s∗(x)=∑j=0ncjφj(x)，使得 ∥f(x)−s∗(x)∥22最小。
- 通过解法方程组 GC=F求得系数 C=(c0,…,cn)T。其中 Gram 矩阵 G的元素为 Gij=(φi,φj)，右端项 Fi=(f,φi)。
- 若基函数 {φj(x)}是带权正交的，则法方程组为对角阵，求解简化。
练习：求 f(x)=x3在区间 [−1,1]上关于权函数 ρ(x)=1的一次最佳平方逼近多项式。
4. 数值积分与微分 
4.1 数值积分 
- 代数精度：若求积公式对不超过 m次的多项式精确成立，但对 m+1次不成立，则称其具有 m次代数精度。
- Newton-Cotes 公式：对于 n+1个等距节点：
  - 当 n为偶数时，代数精度为 n+1（如 Simpson 公式精度为 3）。
  - 当 n为奇数时，代数精度为 n（如梯形公式精度为 1）。
例题：确定求积公式 ∫−11f(x)dx≈[f(−1)+2f(x1)+3f(x2)]/3中的参数 x1,x2，使其代数精度尽量高，并指出所得公式的代数精度。
4.2 数值微分 
常用公式及余项：
- 向前差商：f′(xi)≈hf(xi+h)−f(xi)，余项 −2hf′′(ξ)
- 向后差商：f′(xi)≈hf(xi)−f(xi−h)，余项 2hf′′(ξ)
- 中心差商：f′(xi)≈2hf(xi+h)−f(xi−h)，余项 −6h2f(3)(ξ)
5. 插值法 
5.1 Lagrange 插值 
插值多项式：
Ln(x)=i=0∑nli(x)f(xi),li(x)=j=i∏xi−xjx−xj
余项：
Rn(x)=(n+1)!f(n+1)(ξ)ωn+1(x),ωn+1(x)=k=0∏n(x−xk)
5.2 Newton 插值 
插值多项式：
Nn(x)=f[x0]+f[x0,x1](x−x0)+⋯+f[x0,…,xn]k=0∏n−1(x−xk)
余项：Rn(x)=f[x,x0,…,xn]ωn+1(x)
5.3 Hermite 插值 
要求函数值及导数值匹配。可用重节点差商表构造。
例题：求一个四次插值多项式 H(x)，满足：
- x=0时，H(0)=−1,H′(0)=−2
- x=1时，H(1)=0,H′(1)=10,H′′(1)=40
- 并写出余项表达式。
6. 常微分方程数值解法 
考虑初值问题：dxdy=f(x,y),y(x0)=y0。
Xn = X0 + n·h
- 显式 Euler 法：yn+1=yn+hnf(xn,yn)，局部截断误差 O(h2)，一阶精度。
- 隐式 Euler 法：yn+1=yn+hnf(xn+1,yn+1)，同样一阶精度。
- 改进 Euler 法（梯形公式）：二阶精度。
- 局部截断误差：若 En+1=O(hp+1)，则方法为 p阶。

三、 线性和非线性规划部分 (约20%) 
1. 线性规划的单纯形法 
标准形式：mincTx，s.t. Ax=b,x≥0。
- 单纯形表：通过迭代旋转运算，找到最优解。
- 大M法：处理人工变量，将目标函数修改为 mincTx+M∑xa（xa为人工变量）。
例题：
mins.t.−3x1−x23x1+3x2+x3=304x1−4x2+x4=162x1−x2≤12xj≥0,j=1,2,3,4
引入松弛变量 x5化为标准型，用单纯形法求解。最终得最优解 xˉ=(7,3,0,0,1)T，最优值 fmin=−24。
2. 对偶理论 
写出对偶模型：
- 原问题为 min，则对偶问题为 max。
- 约束条件的符号方向与对偶变量的符号限制相对应。
例题：对线性规划问题：
minz=4x1+x2+2x3+3x4⎩⎨⎧6x1+x2+2x3+5x4≥15−2x1+7x2−x3+x4≥2x1+5x2+4x3+2x4≥0x1,x2,x3,x4≥0
写出其对偶规划模型。
3. 原问题与对偶问题的最优解关系 
例题：已知原问题最优单纯形表如下（其中 x4,x5是松弛变量），求对偶问题的最优解。
maxz=3x1+x2+5x3⎩⎨⎧6x1+3x2+5x3≤453x1+4x2+5x3≤30xj≥0
关键：对偶问题的最优解可由原问题最终单纯形表中松弛变量的检验数（取相反数）以及非基变量的检验数（取相反数）读出。
4. 非线性规划的梯度法（最速下降法） 
步骤：
1. 计算当前点的梯度 ∇f(x(k))。
2. 确定搜索方向 d(k)=−∇f(x(k))。
3. 沿方向进行一维线搜索：minλ≥0f(x(k)+λd(k))，求得步长 λk。
4. 更新迭代点：x(k+1)=x(k)+λkd(k)。
例题：用最速下降法求解 minf(x)=x12−2x1x2+4x22+x1−3x2，取初始点 X(0)=(1,1)T，迭代一次。
5. KKT 条件求解非线性规划 
用于求解带约束的非线性规划问题。KKT 条件是局部最优解的一阶必要条件（对于凸规划，也是充分条件）。
例题：求解非线性规划：
\begin{array}{ll}
\min & (x_1 - \frac{4}{9})^2 + (x_2 - 2)^2 \\
\text{s.t.} & x_2 - x_1^2 \ge 0 \\
& x_1 + x_2 \le 6 \\
& x_1, x_2 \ge 0
\end{array}
\解答提示：写出 KKT 条件方程组，通过讨论约束的活跃集，求得满足所有条件的解。该问题是凸规划，故 KKT 点即为全局最优解。解得 \(x_1 = \frac{3}{2}, x_2 = \frac{9}{4}\)，对应的乘子 \(w_1 = \frac{1}{2}, w_2 = w_3 = w_4 = 0\)。
