### 二、 数值分析部分 (约40%) 
 1. 线性方程组迭代法 
将方程组 Ax=b改写为 x=Bx+f，构造迭代格式：x(k+1)=Bx(k)+f。
- 收敛性判定：
  - 定理 4.1.1：迭代法收敛的充要条件是谱半径 ρ(B)<1。
  - 充分条件：若矩阵范数 ∥B∥<1，则迭代法收敛。
- 收敛速度：定义渐近收敛速度为 R(B)=−lnρ(B)。
2. 非线性方程迭代法 
将方程 f(x)=0化为不动点形式 x=φ(x)，构造迭代格式：xk+1=φ(xk)。
- 收敛性定理（定理 4.3.2）：若迭代函数 φ(x)在区间 [a,b]上满足压缩条件，则对任意初值 x0∈[a,b]，迭代序列收敛。
- 收敛阶判定（定理 4.3.3）：若 φ′(x∗)=0，且 φ(p)(x∗)=0，则迭代法为 p阶收敛。
3. 函数的最佳平方逼近 
在函数空间 C[a,b]的子空间 Φ=span{φ0(x),…,φn(x)}中，寻找 s∗(x)=∑j=0ncjφj(x)，使得 ∥f(x)−s∗(x)∥22最小。
4. 数值积分与微分 
4.1 数值积分 
- 代数精度：若求积公式对不超过 m次的多项式精确成立，但对 m+1次不成立，则称其具有 m次代数精度。
- Newton-Cotes 公式：对于 n+1个等距节点：
  - 当 n为偶数时，代数精度为 n+1（如 Simpson 公式精度为 3）。
  - 当 n为奇数时，代数精度为 n（如梯形公式精度为 1）。
4.2 数值微分 
常用公式及余项：
- 向前差商：f′(xi)≈hf(xi+h)−f(xi)，余项 −2hf′′(ξ)
- 向后差商：f′(xi)≈hf(xi)−f(xi−h)，余项 2hf′′(ξ)
- 中心差商：f′(xi)≈2hf(xi+h)−f(xi−h)，余项 −6h2f(3)(ξ)
